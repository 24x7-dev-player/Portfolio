üöÄ SQL Grouping + Sorting:

  • Commands: Shorting, Groupby, Having
  • Functions: Order by, Limit, Desc, Group by, Having

              -- Select customers ordered by last name (ascending)
              SELECT CustomerID, FirstName, LastName, Email
              FROM Customers
              ORDER BY LastName ASC;              
            

              -- Group customers by country and count the number in each country
              SELECT Country, COUNT(*) AS CustomerCount
              FROM Customers
              GROUP BY Country;              
            

              -- Group customers by country  with more than 10 customers
              SELECT Country, COUNT(*) AS CustomerCount
              FROM Customers
              GROUP BY Country
              HAVING COUNT(*) > 10;
