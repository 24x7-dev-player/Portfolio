üöÄ SQL Grouping + Sorting:

  • Commands: Shorting, Groupby, Having
  • Functions: Order by, Limit, Desc, Group by, Having

              import pandas as pd

              # Sample data
              data = {
                  'salesperson': ['Alice', 'Bob', 'Alice', 'Bob', 'Charlie', 'Charlie', 'David', 'David'],
                  'region': ['North', 'North', 'South', 'South', 'North', 'South', 'North', 'South'],
                  'sales': [250, 200, 150, 300, 450, 500, 300, 250]
              }
              
              df = pd.DataFrame(data)
              
              # Display the DataFrame
              print("Original DataFrame:")
              print(df)
              
              # Sorting
              sorted_df = df.sort_values(by='sales', ascending=False)
              print("\nDataFrame sorted by sales in descending order:")
              print(sorted_df)
              
              # Grouping
              grouped_df = df.groupby('salesperson').sum().reset_index()
              print("\nGrouped DataFrame by salesperson with total sales:")
              print(grouped_df)
              
              # Having
              filtered_df = grouped_df[grouped_df['sales'] > 400]
              print("\nFiltered DataFrame with total sales greater than 400:")
              print(filtered_df)

üöÄ SQL Joins & Operations:

  • Types of Joins: Cross join, Inner join, Right join, Full Outer join, Left Join, self join
  • Operations: Union, Union all, Intersect, except, Joining more than one column, FIlter Columns
  • Keywords: Join, group, cross, left, right, on, full, in 

              -- INNER JOIN: 
              SELECT employees.*, departments.*
              FROM employees
              INNER JOIN departments ON employees.department_id = departments.department_id;
              
              -- LEFT JOIN (or LEFT OUTER JOIN): 
              SELECT employees.*, departments.*
              FROM employees
              LEFT JOIN departments ON employees.department_id = departments.department_id;
              
              -- RIGHT JOIN (or RIGHT OUTER JOIN): 
              SELECT employees.*, departments.*
              FROM employees
              RIGHT JOIN departments ON employees.department_id = departments.department_id;
              
              -- FULL JOIN (or FULL OUTER JOIN): 
              SELECT employees.*, departments.*
              FROM employees
              FULL JOIN departments ON employees.department_id = departments.department_id;
              
              -- CROSS JOIN: 
              SELECT employees.*, departments.*
              FROM employees
              CROSS JOIN departments;
