üöÄFeature Scaling :

  • algorithm Affected: Gradient Descent, SVM, Distance Based, Dimenionality Reduction
  • Not Algorithm Affected: Tree Based, Naive Based
  • Not Algorithm Affected: Normalization and Standarization
  • Standardization: Adjusts the values to have a mean of 0 and a standard deviation 1
  • Min-Max Scaling: Rescales the values to a range between 0 and 1.
  • Robust Scaler: Scales the data using statistics that are robust to outliers.
  • Max Absolute Scaler: Scales the data so that the maximum absolute value of each feature is 1.
  • L2/L1 Normalization: Rescales the values of each sample to have a unit norm (sum of squares or absolute values).

üöÄOutlier Detection :

  • Impact: Preprocessing, Visualization, Model Performance
  • Removal: Deleting outliers from the dataset to prevent them from affecting analysis.
  • Capping: Limiting the extreme values of outliers to a specified threshold.
  • Discretization: Grouping continuous data into categories or bins.
  • Impute: Replacing missing or outlier values with estimated ones.
  • Zscore, IQR: Statistical methods for detecting outliers based on the data's mean and spread.
  • Isolation, LOF, OneClass SVM, DBSCAN: Machine learning algorithms for outlier detection based on data clustering and density.
  • Box plot, Scatter plot: Visualization techniques to visually identify outliers in a dataset.
  • key Concepts: Univariate Technique(Column Wise)(IQR, ZScore), MultiVaarient Outlier Detection(Raw wise)(Isolation Forest, LOF, Oneclass SVM, DBSCAN, KNN),
